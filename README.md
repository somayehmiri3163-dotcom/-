# Parking Management System

A simple Java-based Parking Management System for managing vehicle entry, exit, parking slots, and fee calculation.

## Features
- User login (Admin / Operator)
- Vehicle registration
- Entry and exit management
- Parking slot allocation
- Fee calculation
- Reports

## Technologies
- Java (OOP)
- UML (PlantUML)
- GitHub

## Project Structure
- /docs → UML diagrams (puml files)
- /diagrams → exported images (PNG)
- /src → Java source code (optional)
- schema.sql → database structure
